# Sugar Dockerized Sugar version 13
This repository will help you deploy a Docker based development full stack for Sugar version 13

## Stacks available
* php82-local-build.yml - Main reference stack - Single web server
    * PHP - 8.2.2
    * Apache - 2.4
    * MySQL - 8.0.32
    * Elasticsearch - 8.4.3
    * Redis - latest
